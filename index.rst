Documentation Oxygen
====================

Cette documentation a pour but d'expliquer le fonctionnement d'Oxygen (framework Open Source basé sur Symfony2). 
C'est donc une documentation technique axée sur les mécanismes et les développements du framework Oxygen. 

**Cette documentation n'a pas pour but d'expliquer comment fonctionne Symfony2**. Nous vous renvoyons donc 
vers la `documentation officielle de Symfony2 <http://www.symfony.com/>`_.

La documentation est écrite en parallèle des développements et des améliorations du framework. 
N'hésitez pas à faire remonter vos remarques en contactant l'équipe via support@oxygen.soletic.org.

Bonne lecture,

Oxygen Core Team

Pourquoi Oxygen ?
-----------------

Aujourd'hui, de nombreux logiciels Open Source permettent de créer des applications web de toutes natures 
(CMS, eCommerce, Réseau social, etc.). Cependant, dès qu'il s'agit de créer une application spécifique 
à une problématique métier centrée sur l'activité d'une entreprise, ces solutions Open Source manquent 
de souplesse et s'adapte difficilement à ces problématiques. Le détournement d'une solution *clé en main* 
peut entraîner :

* des baisses des performances,
* une ergonomie peu adaptée,
* l'impossibilité de répondre à une nécessité fonctionnelle à moindre coût,
* une maintenance complexe,

Oxygen a était créé pour répondre aux besoins des développeurs web souhaitant créer rapidement des applications 
métiers. Basé sur Symfony2, Oxygen vous propose des bundles pour aller plus vite dans vos développements que 
le projet soit "simple" ou "complexe".

Oxygen offre des fonctionnalités universelles et est donc composé d'un noyau d'application embarquant le strict 
nécessaire pour faire tourner une application complétée par des modules extensibles (*bundles*). Chaque bundle Oxygen 
a un seul objectif et est composé de fonctionnalités permettant de le remplir.

**Il n'est donc plus de nécessaire de trouver un compromis entre ce que permet une solution clé en main, 
l'exigence d'un métier et le budget dont vous disposez.**
